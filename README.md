# Anurag Narula — Portfolio Website

A personal portfolio website built with React and Vite, showcasing my experience, projects, and skills as a Software Engineer.

🌐 **Live:** [anuragnarula.vercel.app](https://anuragnarula.vercel.app/)

---

## Sections

- **Hero** — Introduction and current role
- **About** — Background, interests, and personal insights
- **Experience** — Professional experience at FNZ and MakeMyTrip
- **Projects** — Featured personal and professional projects
- **Technologies** — Tools and technologies I work with
- **Contact** — Ways to get in touch

# Build for production
